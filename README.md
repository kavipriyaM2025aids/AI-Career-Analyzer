# AI Career Analyzer

An AI-powered web application that analyzes user skills and interests to suggest suitable career paths and provide personalized recommendations.

## Features
- Career recommendation system
- Skill analysis
- Career matching
- Personalized guidance dashboard
- Interactive web interface

## Technologies Used
- Python
- FastAPI
- HTML
- CSS
- JavaScript

## How to Run

### 1. Install dependencies

```bash
pip install fastapi uvicorn
```

### 2. Start the server

```bash
uvicorn app:app --reload
```

### 3. Open in browser

```text
http://127.0.0.1:8000
```
